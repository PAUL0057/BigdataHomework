样本容量的确定

因为要观察样本质量和样本容量之间的关系，在样本量较小的时候，样本质量随着样本量的变化较大；而当样本量较大接近样本总量时，样本质量的变化较小。因此在选取样本容量时，我们通过指数函数构建如下样本数，选取68个抽样样本数。

#通过指数分布选取68个样本容量
x = seq(10,20,0.1)
y = 2^(x)
sampl = round(y)[1:68]
sample_num = as.matrix(sampl)

构建随机抽样&分层抽样&计算样本质量得分的函数

构建随机抽样函数,该函数返回被抽到的income数据。

#构建随机抽样的函数，该函数返回被抽到的income数据。
sample1 = function(x){
  y = sample(income,x)
  y = c(y,rep(NA,sampl[length(sampl)]-length(y)))
  return(y)
}

构建分层抽样的函数,该函数(sample_result_func)返回被抽到的income数据。

sample_result_func = function(){
z = NULL
for (i in 1:68) { y = NULL
  for (j in 1:16) {
    y1 = sample2(i,j)
    y = c(y,y1)}
  y = c(y,rep(NA,(sampl[68]-length(y))))
  z = cbind(z,y)}
return (z)}

构建计算样本质量得分的函数，该函数返回抽样得分。

#构建计算样本质量的函数，该函数返回抽样得分。
get_score = function(sample_result){
  income_sample = cut(na.omit(sample_result),breaks = my_break)
  freq1 = table(income_sample)/length(na.omit(sample_result))+1e-15#防止生成Inf
  score = exp(-sum((freq1-freq)*(log(freq1/freq))))
  return(score)
}

简单随机抽样

对每一个不同的样本容量进行50次随机抽样，通过取50次随机抽样得分的均值，避免单词抽样所带来的偶然性。并画出随样本量变化的样本质量得分变化，其中y轴是样本质量，x轴是样本容量，可以看出在样本量较小的时候，样本量的小幅提升可以带来样本质量相对大幅度的上升。

scores = NULL

分层抽样

根据总体的income频数表和样本容量，来确定每个组所需要的样本个数，在每个组内进行简单随机抽样，取50次分层抽样得分的均值，避免单次抽样带来的偶然性结果。并画出随样本量变化的样本质量得分变化。可以看出分层抽样的结果相对随机抽样来说，在小样本的表现上更加优异。

两个抽样方法的质量得分比较

将两个抽样方法的图重合，可以发现在小样本容量是，分层抽样明显优于随机抽样。像样本量同为2048时，随机抽样的质量为0.985，而分层抽样是0.998。但是随机抽样随着样本量的增加质量也增长的很快。 整体来看，分层抽样明显优于随机抽样,当我们选择样本质量为0.999时,分层抽样只要3104个样本，而随机抽样则需要23130个。随机抽样的成本会更高。 由此可见，分层抽样的样本更能够代表全面的总体数据。
